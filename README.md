# Finance Manager by Erik Georgiev
This is a finance manager which allows you to store your transactions in a file
and easily view them.

Features:
1. Add Transaction
2. Add Frequent Transaction
3. Use Frequent Transaction
4. View Transactions
5. Clear All Transactions

How to run program:
- copy the entire CPT folder onto your device.
- go to CPT\src\FinanceManager.java
- click run (or press F5)
- follow instructions in the terminal
